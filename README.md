# 🏥 Hospital Emergency Room Dashboard | Power BI Project

🚑 **Turning healthcare data into actionable insights with Power BI!**

I'm excited to share one of my recent **Power BI Dashboard** projects – a **Hospital Emergency Room Dashboard** designed to help healthcare teams monitor patient flow, operational efficiency, and service quality.

### 📊 Dashboard Highlights

✅ **Monthly View**

* Total Patients
* Average Wait Time
* Patient Satisfaction Score
* Patients Referred
* Admission Status
* Patients Seen Within 30 Minutes
* Patient Distribution by Age Group
* Department Referrals
* Patient Race Analysis
* Daily Patient Trends
* Wait Time Heatmap

✅ **Consolidated View**

* Complete hospital performance across the selected date range
* KPI comparison
* Trend analysis
* Overall emergency department performance

✅ **Patient Details**

* Detailed patient-level records
* Patient ID
* Gender
* Age
* Admission Date
* Race
* Wait Time
* Department Referral
* Easy filtering with interactive slicers

### 🛠️ Tools Used

* 📌 Power BI
* 📌 Power Query
* 📌 DAX
* 📌 Data Modeling
* 📌 Interactive Slicers
* 📌 KPI Cards
* 📌 Bar Charts
* 📌 Donut Charts
* 📌 Heatmaps
* 📌 Matrix Visuals

### 💡 Key Business Insights

* Identify peak patient arrival days.
* Track emergency room wait times.
* Monitor patient satisfaction trends.
* Analyze admission vs. non-admission rates.
* Understand referral patterns across departments.
* Improve operational efficiency with real-time insights.

This project demonstrates how **data visualization** can support faster decision-making and improve healthcare operations
